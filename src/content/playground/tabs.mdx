# Tabs

Tabs are UI components that display one content panel at a time, allowing users to switch between different sections or content areas within the same interface.

```tsx
<Tabs defaultValue="JavaScript">
  <TabList items={['JavaScript', 'TypeScript']} />
  <TabContent value="JavaScript">
    JavaScript content
  </TabContent>
  <TabContent value="TypeScript">
    TypeScript content
  </TabContent>
</Tabs>
```
