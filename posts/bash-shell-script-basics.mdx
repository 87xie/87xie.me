## Hashbang

indicate an interpreter

```bash
#! /bin/bash
```

## source Command

- `source`: 在當前的程序下執行腳本，可以縮寫成 `.`
- `sh`: 給予一個新的程序執行腳本

```bash
cat << EOF > hello.sh
#! /bin/bash

echo \$NAME
EOF
```

```bash
NAME=87xie
source ./hello.sh
# 87xie
. ./hello.sh
# 87xie
sh ./hello.sh
#
```

## Arguments

執行 shell script 並傳入參數

hello.sh

```bash
#! /bin/bash

GREETING_WORD=$1
NAME=$2

echo "${GREETING_WORD} ${NAME}"
# hi 87xie
echo ${0}
# ./hello.sh
echo ${#}
# 2
echo ${@}
# hi 87xie
echo ${*}
# hi 87xie
```

```bash
./hello.sh hi 87xie
```

default arguments

| syntax | description             |
|--------|-------------------------|
| $0     | the file name           |
| $#     | the number of arguments |
| $@     | "$1" "$2" "$3" … "$\{N}" |
| $*     | "$1c$2c$3c$4c … $\{N}" <br /> where c is the first character of the value of the IFS variable. If IFS is unset, the parameters are separated by spaces. |
| $?     | the exit status of last command |

## If Statements

- `read -p`: prompt
- `fi` 表示結束條件判斷
- `[]` 頭尾需要空格

```bash
read -p "type a number: " num

if [ $num -le 5 ]; then
  echo "in if block"
elif [ $num -le 10 ]; then
  echo "in else if block"
else
  echo "in else block"
fi
```

```bash

case $1 in
```

### Variables Scope

關鍵字 `local` 將變數作用域指定在 function block

```bash
#! /bin/bash

greeting() {
  local locale=$1
  local en_us_greeting="hi there"
  local zh_tw_greeting="你好"

  case $locale in
    "zh-tw")
      echo $zh_tw_greeting
      ;;
    "en-us")
      echo $en_us_greeting
      ;;
    *)
      echo "hello"
      ;;
  esac
  
  return 0
}
```
## Arrays

indexed array

```bash
#! /bin/bash

my_arr=(foo bar)

echo ${my_arr[*]}
# foo bar (all of item in array)
echo ${!my_arr[@]}
# 0 1 (all of indexes)
echo ${#my_arr[*]}
# 2 (the length of array)
```

associative array

- 使用關鍵字 `declare -A` 宣告

```bash
#! /bin/bash

declare -A my_arr
my_arr=([foo]=hello [bar]=world)

echo ${my_arr[*]}
# hello world
echo ${!my_arr[@]}
# foo bar
```

## For Loops

```bash
#! /bin/bash

my_arr=(foo bar)

for el in ${my_arr[*]}
do
  echo $el
done
```

break

```bash
#! /bin/bash

my_arr=(foo bar)

for index in "${!my_arr[@]}"
do
  if (( $index >= 1)); then
    break
  fi
  echo "index is $index"
done
```

## While

```bash
#! /bin/bash

# let "NUM_TO_GUESS = ${RANDOM} % 10 + 1"
NUM_TO_GUESS=$(( $RANDOM % 10 + 1 ))
GUESSED_NUM=0

echo "guess a number between 1 and 10"

while [ $NUM_TO_GUESS -ne $GUESSED_NUM ]
do
  read -p "your guess: " GUESSED_NUM
done

echo "you got it!"
```
